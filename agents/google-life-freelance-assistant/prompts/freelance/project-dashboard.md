# Dashboard de proyectos freelance

```text
Arma un dashboard de proyectos freelance usando la informacion disponible.

Inputs:
- Clientes/proyectos: [projects]
- Correos relevantes: [emails]
- Eventos relevantes: [calendar_events]
- Documentos relevantes: [drive_docs]
- Fecha actual: [today]

Output:
Tabla por proyecto:
- Cliente
- Proyecto
- Estado
- Ultima novedad
- Proximo entregable
- Fecha importante
- Riesgo
- Proxima accion

Luego agrega:
1. Proyectos que necesitan atencion esta semana
2. Clientes esperando respuesta
3. Entregables en riesgo
4. Posibles oportunidades comerciales

Restricciones:
- No inventes estados.
- Si el estado es inferido, marcalo.
- No confirmes compromisos sin aprobacion del usuario.
```
