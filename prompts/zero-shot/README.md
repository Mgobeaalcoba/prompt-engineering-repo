# Zero-shot prompting

Esta tecnica pide una tarea directamente, sin ejemplos previos. Funciona bien cuando la instruccion es clara y el output esta definido.

## Ejemplos incluidos

- `resumen-ejecutivo.md`
- `plan-accion.md`
- `checklist-lanzamiento.md`
- `explicacion-simple.md`
- `revision-riesgos.md`
