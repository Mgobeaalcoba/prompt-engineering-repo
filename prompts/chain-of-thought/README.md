# Chain-of-thought prompting

Esta tecnica busca mejorar tareas complejas pidiendo que el modelo organice su analisis antes de responder. Para uso practico, se recomienda solicitar criterios, pasos de evaluacion o justificacion breve, en lugar de exigir razonamiento interno extenso.

## Ejemplos incluidos

- `decision-compra-software.md`
- `diagnostico-problema.md`
- `priorizacion-roadmap.md`
- `analisis-causa-raiz.md`
- `comparacion-alternativas.md`
