# Drive package

Material preparado para subir a Google Drive como soporte de la Etapa 2: Prompt Engineering.

## Archivos

- `prompt_catalog_etapa2.csv`: matriz de prompts con consignas, tecnicas, objetivos, variables, outputs, errores a evitar y criterios de aceptacion.
- `manual-interaccion-asistente-personal.html`: manual de interaccion para pedirle cosas al asistente de manera efectiva.
- `drive-index.html`: indice organizador del concepto, prompts base, prompts especializados, tecnicas avanzadas, manual y evaluaciones.

## Version subida a Drive

- Google Sheet: Etapa 2 - Catalogo de Prompts - Asistente Personal Google
- Google Doc: Manual de Interaccion - Asistente Personal Google
- Google Doc: Carpeta Logica - Asistente Personal Google - Indice
