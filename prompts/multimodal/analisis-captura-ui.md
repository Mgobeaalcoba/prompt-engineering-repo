# Analisis de captura de interfaz

## Tecnica

Multimodal prompting

## Prompt

```text
Analiza la captura de pantalla adjunta de una interfaz digital.

Objetivo:
Identificar problemas de claridad, jerarquia visual, accesibilidad y usabilidad.

Revisa:
- Elementos principales visibles
- Accion primaria esperada
- Posibles puntos de confusion
- Problemas de contraste, tamanos o espaciado
- Recomendaciones priorizadas

Formato de salida:
1. Resumen breve
2. Hallazgos en tabla: problema, impacto, severidad, recomendacion
3. Top 3 cambios prioritarios

No inventes elementos que no se vean en la imagen.
```

## Variables editables

| Variable | Descripcion |
| --- | --- |
| Imagen adjunta | Captura de pantalla a analizar |

## Criterios de calidad

- Se basa en lo visible.
- Prioriza problemas accionables.
- Considera accesibilidad.
