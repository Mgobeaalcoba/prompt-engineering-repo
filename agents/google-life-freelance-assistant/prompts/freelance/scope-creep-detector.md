# Detector de scope creep

```text
Analiza comunicaciones de un proyecto freelance para detectar posible expansion de alcance.

Inputs:
- Hilos de Gmail: [threads]
- Documentos de alcance o propuesta: [scope_docs]
- Eventos o notas de reunion: [calendar_or_notes]

Busca:
- Nuevos pedidos no contemplados
- Cambios de prioridad
- Entregables adicionales
- Urgencias no planificadas
- Ambiguedades de responsabilidad

Output:
1. Senales de scope creep
2. Evidencia textual breve
3. Impacto potencial
4. Preguntas para aclarar
5. Borrador de respuesta cuidadosa para ordenar alcance

Restricciones:
- No acuses al cliente.
- Usa tono colaborativo.
- No cites contratos si no estan disponibles.
```
