# Explicacion simple de un concepto

## Tecnica

Zero-shot prompting

## Prompt

```text
Explica el concepto "[concepto]" para una persona sin experiencia previa.

Estructura:
1. Definicion en una frase
2. Analogia cotidiana
3. Ejemplo practico
4. Errores comunes de interpretacion
5. Resumen en tres bullets

Restricciones:
- Evita tecnicismos innecesarios.
- Si usas un termino tecnico, definilo.
- Maximo 500 palabras.
```

## Variables editables

| Variable | Descripcion |
| --- | --- |
| `[concepto]` | Tema a explicar |

## Criterios de calidad

- Es entendible para principiantes.
- Usa ejemplos concretos.
- No simplifica de forma enganosa.
