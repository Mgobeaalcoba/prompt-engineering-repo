# Codigo de conducta

Este proyecto busca ser un espacio colaborativo, respetuoso y util.

## Principios

- Tratar a las personas con respeto.
- Hacer criticas sobre ideas, no sobre personas.
- Explicar sugerencias con claridad.
- Evitar contenido discriminatorio, abusivo o deliberadamente enganoso.
- No publicar informacion privada o sensible.

## Reportes

Si detectas una conducta inapropiada, abre un issue con contexto suficiente o contacta a la persona mantenedora del proyecto.
