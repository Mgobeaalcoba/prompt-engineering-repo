# Biblioteca de prompts

Cada carpeta contiene ejemplos listos para copiar y adaptar. Los archivos usan Markdown para que GitHub los muestre de forma legible.

## Indice por tecnica

| Carpeta | Tecnica | Cantidad de ejemplos |
| --- | --- | --- |
| `few-shot/` | Few-shot prompting | 5 |
| `chain-of-thought/` | Chain-of-thought prompting | 5 |
| `zero-shot/` | Zero-shot prompting | 5 |
| `multimodal/` | Multimodal prompting | 5 |
| `rol-contexto-tarea-output-restricciones-ejemplos/` | ROL + Contexto + Tarea + Output + Restricciones + Ejemplos | 5 |

## Sugerencia de uso

Para crear un nuevo prompt, copia `templates/prompt-template.md`, completa las secciones y guardalo en la carpeta de tecnica correspondiente.
