# Few-shot prompting

Esta tecnica incluye ejemplos concretos dentro del prompt para que el modelo imite estructura, estilo, tono o criterio de clasificacion.

## Ejemplos incluidos

- `clasificacion-feedback-clientes.md`
- `reescritura-tono-marca.md`
- `extraccion-datos-correo.md`
- `generador-titulos-blog.md`
- `evaluacion-respuestas-soporte.md`
