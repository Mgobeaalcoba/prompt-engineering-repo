# Borrador de respuesta de Gmail

```text
Redacta un borrador de respuesta para el siguiente correo o hilo.

Correo/hilo:
[thread]

Objetivo del usuario:
[user_goal]

Contexto adicional:
[context]

Estilo:
- Claro
- Amable
- Profesional
- Breve

Output:
1. Asunto sugerido si aplica
2. Borrador listo para revisar
3. Puntos que requieren confirmacion del usuario
4. Riesgos o ambiguedades

Restricciones:
- No confirmes fechas, precios, entregables ni compromisos si no estan explicitamente aprobados.
- Si falta informacion, usa placeholders entre corchetes.
- No envies el correo. Solo redacta el borrador.
```
