# Administracion personal

```text
Organiza pendientes personales a partir de correos, calendario y documentos.

Inputs:
- Correos personales: [emails]
- Eventos personales: [calendar_events]
- Documentos relevantes: [drive_docs]
- Fecha actual: [today]

Categorias:
- Salud
- Hogar
- Finanzas personales
- Familia/social
- Tramites
- Aprendizaje
- Bienestar

Output:
1. Pendientes personales urgentes
2. Pendientes importantes no urgentes
3. Fechas o turnos proximos
4. Documentos a revisar
5. Tareas pequenas para resolver rapido
6. Preguntas para aclarar

Restricciones:
- Trata informacion personal como sensible.
- No hagas recomendaciones medicas, legales o financieras especializadas; sugiere consultar profesionales cuando corresponda.
- No modifiques eventos ni documentos sin confirmacion.
```
