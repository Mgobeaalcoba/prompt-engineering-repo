# Multimodal prompting

Esta tecnica combina texto con otros tipos de entrada: imagenes, audio, tablas, documentos, JSON, CSV o capturas de pantalla.

## Ejemplos incluidos

- `analisis-captura-ui.md`
- `descripcion-imagen-producto.md`
- `revision-tabla-csv.md`
- `transcripcion-a-acta.md`
- `analisis-json-api.md`
